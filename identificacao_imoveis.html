Identificação de Imóveis para Compra e Revenda

Este projeto envolveu a identificação de imóveis abaixo do preço médio de venda e a definição do preço ideal de revenda. Utilizei uma análise exploratória de dados em Python para alcançar esses objetivos.

Ferramentas Utilizadas:

  • Python, Pandas, Numpy e Seaborn
  • Anaconda, Pycharm e Jupyter Notebook
  • Mapas interativos com Plotly e Folium
  • Heroku Cloud
  • Streamlit Python Framework Web
