Cury Company - Análise de KPIs Estratégicos

Nesse projeto, os conceitos de Programação em Python, manipulação de dados, pensamento estratégico e lógica de negócio, junto com ferramentas de desenvolvimetno web como o Streamlit e Github, foram usados para desenvolver um painel gerencial com as principais métricas de uma empresa marketplace de delivery de comida na India. Preço dos Combustíveis

Coletamos dados de preços de combustíveis de 2021 do site oficial e realizamos o pré-processamento, que incluiu limpeza e transformação dos dados. Aplicamos o algoritmo K-médias para agrupar os dados em clusters regionais e testamos vários modelos de aprendizado de máquina, como Regressão Linear e Random Forest, para prever os preços e avaliar seu desempenho. O agrupamento com K-médias revelou padrões regionais nos preços dos combustíveis. O modelo Random Forest foi o mais eficaz, com MSE de 0.052 e R² de 0.92. A validação cruzada confirmou a robustez dos resultados, fornecendo insights valiosos para estratégias futuras no mercado de combustíveis.

Fome Zero Growth Dashboard

Desenvolvemos o Fome Zero Growth Dashboard coletando dados dos restaurantes e usando Python com Streamlit para criar uma aplicação web interativa. Aplicamos Pandas para análise de dados, Plotly para gráficos interativos, Folium para visualização geoespacial e Pillow para imagens, resultando em um painel online acessível. O painel online "Painel Fome Zero" oferece uma visão detalhada do desempenho dos restaurantes, permitindo análises por cidade, país e tipo de culinária, e ajuda na tomada de decisões estratégicas para expansão e melhoria.
